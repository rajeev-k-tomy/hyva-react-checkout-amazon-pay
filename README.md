# Hyva React Checkout Amazon Pay
This is a Magento 2 module that supports Hyv채 React Checkout Amazon Pay integration.

## Installation

1. Using composer:
    ```
    composer require rajeev-k-tomy/magento2-react-checkout-amazon-pay
    ```

## What it does

1. It injects necessary amazon pay configuration into `CheckoutConfigProvider` of hyv채 react checkout.
2. It removes certain phtml files which are no longer compatible with hyv채 react checkout. (Note: only on hyv채 themes)
